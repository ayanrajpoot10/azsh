# azsh

Access Azure Cloud Shell from your local terminal.

[Azure Cloud Shell](https://learn.microsoft.com/en-us/azure/cloud-shell/overview) is a browser-based environment with common tools (az, kubectl,
docker, etc.) preinstalled. This tool brings that shell to your terminal so
you don't need to install the az CLI locally (it takes too much space, see [azure-cli#7387](https://github.com/Azure/azure-cli/issues/7387)) or keep a browser tab
open.

## Install

```
go install github.com/ayanrajpoot10/azsh@latest
```

## Uses

```
azsh
```
